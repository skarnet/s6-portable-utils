The s6-cut program

s6-cut prints selected portions of the lines of its input.

Interface 

     s6-cut [ -b list | -c list | -f list ] [ -n ] [ -s ] [ file ... ]

s6-cut acts as the generic cut utility.

Posixness

s6-cut is suitable as a Single Unix cut program.
