The s6-mkdir program

s6-mkdir creates directories.

Interface 

     s6-mkdir [ -p ] [ -v ] [ -m mode ] [ dir... ]

s6-mkdir acts as the generic mkdir utility.

Extra options

  • -v : write what it does to stderr.

Posixness

s6-mkdir is suitable as a Single Unix mkdir program.
